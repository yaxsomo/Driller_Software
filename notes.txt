//LAMAGE (OP. 5)

MS A S [CoordX] [CoordY] [EPAISSEUR_TOLE] [EMPLACEMENT_OUTIL] [VITESSE_ROTATION_OUTIL] [VITESSE_AVANCE_OUTIL] [PROFONDEUR] [LONGUEUR_PILOTE_OUTIL]





[CoordX] = Coordonnee X de l'emplacement de l'operation dans le Repere -> Format 000000 (ex. 200000 pour decrire un emplacement à 2000,00mm)
[CoordY] = Coordonnee Y de l'emplacement de l'operation dans le Repere -> Format 000000 (ex. 200000 pour decrire un emplacement à 2000,00mm)
[EPAISSEUR_TOLE] = Epaisseur de la tole -> Format 000 (ex. 060 pour decrire une epaisseur de 6 mm ou 100 pour decrire une epaisseur de 10mm)
[EMPLACEMENT_OUTIL] = Emplacement de l'outil -> format 00 (ex. 04 pour decrire l'emplacement 4)
[VITESSE_ROTATION_OUTIL] = Vitesse de rotation de l'outil -> format 0000 (ex. 0800 pour decrire une vitesse de 800 tr/min)
[VITESSE_AVANCE_OUTIL] = Vitesse d'avance de l'outil -> format 000 (ex. 700 pour decrire une vitesse d'avance de 7mm/s ou 090 pour decrire une vitesse d'avance de 0.9mm/s)
[ANGLE_FRAISE] = Angle de la fraise -> format 000 (ex. 090 pour decrire un angle à 90°)
[DIAMETRE_EXTERIEUR] = Diametre exterieur du chanfrein -> format 000 (ex. 150 pour decrire un diametre de 15mm ou 050 pour decrire un diametre de 5mm)
[PAS_OUTIL] = Pas de l'outil -> format 000 (ex. 125 pour decrire un pas de 1.25mm ou 025 pour decrire un pas de 0.25mm)
[PROFONDEUR] = Profondeur -> format 000 (ex. 095 pour decrire une profondeur de 9.5mm)
[LONGUEUR_PILOTE_OUTIL] = Longueur du pilote de l'outil -> format 000 (ex. 065 pour decrire une longueur du pilote de 6.5mm)
