IsiConnect-cpp

CPP lib used to build an IsiConnect object.

Dependancies

Build and/or install :

  • redis-cpp : https://github.com/tdv/redis-cpp
  • natsio-c : https://github.com/nats-io/nats.c
  • boost

To set up (with VS Code)

Import project folder into VS code. Install C++ and CMakeTools extensions run CMake (debug) in the lower vscode toolbar Click the build button, and run directly inside vscode

To set up (terminal)

cd IsiConnect-cpp
mkdir build
cd build
cmake ..
make
./IsiConnect-cpp-test

To create a line and execute it

Auth: Post on http://localhost:3030/api/auth/login

{
"email": "admin@akeros.com", "password": "akeros"
}

Get the token and use it as Bearer token

Line creation: Post on http://localhost:3030/api/missions

{
"name" : "test",
"description": "test line",
"systemActions" : [
{
"name": "Étape 1",
"type": "testRobot",
"status": "READY",
"action": "loadingTask",
"metadata": "{'toto': 1}",
"designatedRobot": "b4:45:6:d7:75:82"
},
{
"name": "Étape 2",
"type": "testRobot",
"status": "READY",
"action": "unloadingTask",
"metadata": "{'toto': 1}",
"designatedRobot": "b4:45:6:d7:75:82"
}
]
}

Launch the line: Empty post on http://localhost:3030/api/scheduler/launch/[id] replacing id with the id returned by the line creation request
