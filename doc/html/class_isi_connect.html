Detailed Description

template<class T>
class IsiConnect< T >

Main object used to create an IsiConnect agent. In order to implement a new IsiAgent, create an object that inherits and implements this IsiConnect class.

IsiConnect Then add jobs with their callbacks and you're good to go !

template<class T >
virtual bool IsiConnect< T >::isAvailable ( )
pure virtual

Method returning the availability status of the IsiConnect object. The availability is the capability of the IsiConnect object to take a job at a given moment. This method is pure virtual and must be implemented in the IsiConnect inheriting object.

Returns
true IsiConnect object is free to take a job order
false IciConnect is bust and cannot take a job order
